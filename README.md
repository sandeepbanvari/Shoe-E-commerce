# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.



# 👟 Shoe-E-commerce

A stylish and responsive **Shoes E-Commerce Web Application** built using **React, Vite, JavaScript, CSS, and Tailwind CSS**.  
This project delivers a modern online shopping experience where users can browse shoes, view product details, add items to cart, and purchase products easily.

---

## 🚀 Project Overview

The Shoe-E-commerce project is designed to simulate a real-world footwear shopping website with a clean UI and smooth navigation. It focuses on responsive design, reusable React components, and interactive shopping features.

---

## ✨ Key Features

### 🏠 Home Page
- Attractive landing page with hero section
- Featured shoe collections
- Trending products showcase
- Smooth navigation menu

### 🛍️ Product Listing
- Display multiple shoe products with images
- Product name, price, and short details
- Clean grid/card layout

### 👟 Product Details Page
- Click any shoe to open separate details page
- Large product image preview
- Full description
- Price display
- **Add to Cart** button
- **Buy Now** button

### 🛒 Cart Functionality
- Add products to cart
- Remove products from cart
- Update cart items
- Total price calculation

### 📱 Responsive Design
- Mobile friendly
- Tablet optimized
- Desktop layout support

---

## 🛠️ Technologies Used

| Technology | Purpose |
|-----------|---------|
| React.js | Frontend UI Development |
| Vite | Fast Build Tool |
| JavaScript | Application Logic |
| Tailwind CSS | Modern Styling |
| CSS3 | Custom Styling |
| React Router DOM | Page Navigation |
| Git | Version Control |
| GitHub | Project Hosting |
